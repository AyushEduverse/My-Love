# Do You Love Me?

A simple and interactive web page asking the classic "Do you love me?" question, with a playful twist on the "No" option.

## Table of Contents

- [Description](#description)
- [Features](#features)
- [How to Run](#how-to-run)

## Description

This project is a lighthearted web application designed to present a simple question: "Do You Love Me?". Users are given two options: "Yes" and "No". The "No" button incorporates a fun JavaScript interaction, making it move randomly across the screen when hovered over, making it difficult to click. This adds a humorous element to the user experience.

## Features

- **Interactive "No" Button:** The "No" button dynamically changes its position on hover, providing a playful challenge.
- **Simple User Interface:** A clean and straightforward design for an engaging experience.
- **Clear Call to Action:** Prompts the user with a direct question.
- **GIF Integration:** Includes an embedded GIF to enhance visual appeal.

## How to Run

To run this project, simply open the `index.html` file in your web browser. No special server setup or build process is required.

1. Clone the repository (if applicable) or download the project files.
2. Navigate to the project directory.
3. Open `index.html` with your preferred web browser.

```bash
# Example if you have cloned the repository
cd path/to/your/project
start index.html # For Windows
open index.html # For macOS
xdg-open index.html # For Linux
```
